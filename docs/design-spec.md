# UI 设计规范 - 高德地图行程规划

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#4A90D9` | 淡蓝色，用于按钮、标题栏、选中态 |
| 主色浅 | `#7AB3E8` | 悬停态、次要元素 |
| 主色深 | `#3A7BC8` | 按钮按下态 |
| 背景 | `#F0F6FC` | 页面背景，极淡蓝 |
| 卡片 | `#FFFFFF` | 卡片、输入框背景 |
| 文字主 | `#333333` | 标题、正文 |
| 文字辅 | `#888888` | 说明文字、提示 |
| 分割线 | `#E8EEF4` | 卡片分割、列表分割 |
| 成功/确认 | `#52C41A` | 已选状态 |
| 警告 | `#FAAD14` | 换乘提醒 |
| 错误 | `#FF4D4F` | 删除按钮 |

## 字体

| 层级 | 大小 | 粗细 | 用途 |
|------|------|------|------|
| 标题 | 20px | 600 | 页面主标题 |
| 副标题 | 18px | 500 | 模块标题 |
| 正文 | 16px | 400 | 卡片内容、输入框文字 |
| 辅助 | 14px | 400 | 地址、说明文字 |
| 小字 | 12px | 400 | 标签、角标 |

- 字体族：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", sans-serif`

## 间距

| 级别 | 大小 | 用途 |
|------|------|------|
| xs | 8px | 图标与文字间距 |
| sm | 12px | 卡片内边距 |
| md | 16px | 模块间距、页面边距 |
| lg | 24px | 大模块间距 |

## 组件样式

### 卡片
- 背景：白色 `#FFFFFF`
- 圆角：12px
- 阴影：`0 2px 8px rgba(0,0,0,0.08)`
- 内边距：16px
- 选中态：左边框 3px `#4A90D9`

### 按钮
- 主按钮（搜索/下一步）：背景 `#4A90D9`，白色文字，高 48px，圆角 24px，全宽
- 次按钮（上一步）：背景白色，边框 `#4A90D9`，文字 `#4A90D9`，高 48px
- 按下态：背景加深 10%

### 输入框
- 背景：`#F5F8FB`
- 圆角：8px
- 边框：1px `#E0E6ED`
- 聚焦态：边框 `#4A90D9`
- 高度：48px
- 字体：16px

### 步骤指示器
- 圆点直径：12px
- 已完成：实心 `#4A90D9`
- 当前：实心 `#4A90D9` + 外圈阴影
- 未完成：空心圆，边框 `#CCCCCC`
- 连接线：2px `#CCCCCC`（已完成段 `#4A90D9`）

### 地图区域
- 占屏幕高度约 40%（手机竖屏约 250-300px）
- 圆角：12px
- 有结果时显示，无结果时隐藏或显示占位

## 布局

- 最大宽度：480px（手机优先，桌面居中显示）
- 页面使用 flexbox 垂直布局
- 顶部固定标题栏（高度 56px）
- 中间内容区可滚动（flex: 1）
- 底部固定导航栏（高度 64px）
